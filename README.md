# Eccomerce de tienda de zapatos
Este proyecto es para una tienda de ropa.

## Tecnologias Utilizadas
React, JavaScript, Css y Html

### Depenedencias implementadas
React-Router-Dom para navegar entre componentes y Firebase para guardar y utilizar documentos en su base de datos.

### Hooks utilizados
UseState para guardar estados; useEffect para llevar a cabo funciones teniendo en cuenta el cambio de un componente; useContext para crear un contexto que tenga funciones, estados y datos que necesite usar en sus componentes hijos, facilitando el paso de informacion y useParams para hacer din√°mica la URL.
